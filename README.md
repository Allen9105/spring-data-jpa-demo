# Spring Data JPA 教程

该仓库用来深度学习 `Spring Data JPA` 项目，如果该系列教程对您有帮助的话，还请点个 ⭐ 给予精神支持！

### 开发环境

- **JDK 1.8 +**

- **Maven 3.6 +**

- **IntelliJ IDEA ULTIMATE 209.3 +** （注意：请务必使用 IDEA 开发，同时保证安装 `lombok ` 插件）

- **MySQL 8.0 +**
